# Persona - Fictional Character Generator

A private, offline-first tool for writers and actors to build fictional characters with psychological depth.

## Features

- **Character Management** - Create, edit, and organize characters with detailed profiles
- **Psychological Index** - Track Self Esteem vs Social Reputation over time (-5 to +5 scale)
- **Visual Arc Charts** - See character psychological journeys as smooth parabolic curves
- **Relationship Networks** - Map and visualize character relationships with interactive graphs
- **Timeline Events** - Track character milestones and story events
- **Character Bible Export** - Generate printable Character Bible PDFs
- **Multi-Project Support** - Manage multiple projects in one app
- **Dark/Light Themes** - Catppuccin-inspired color palette
- **Import/Export** - JSON and CSV support for backup and sharing
- **Mobile Responsive** - Works on phones and tablets

## How to Use

1. Open `index.html` in any modern web browser
2. Register an account (data stored locally in your browser)
3. Start creating characters!

No installation required - runs entirely in your browser.

## Privacy

All data is stored locally in your browser using localStorage. Nothing is sent to any server. Your character data never leaves your device.
